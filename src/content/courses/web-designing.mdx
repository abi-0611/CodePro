## Overview

Web design is about creating layouts that look good and work well on every screen size. This course teaches HTML and CSS the practical way, focusing on responsive structure, spacing, typography, and building a complete website.

### What you’ll learn

- Create clean, semantic HTML structures
- Style pages using modern CSS and layout systems
- Build responsive layouts with Flexbox and Grid
- Apply accessibility and UI best practices

### Why take this course

1. Learn the fundamentals that every frontend developer needs
2. Build a real responsive website and deploy it
3. Develop strong layout intuition and UI clarity
